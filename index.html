Política de Privacidad - SimShockPad

Esta política de privacidad se aplica a la aplicación SimShockPad creada por Pablo Kessler Saiz (en adelante, el "Proveedor de Servicios") como un servicio gratuito. Este servicio se ofrece "TAL CUAL".

Recopilación y uso de información

La aplicación no recopila datos personales del usuario. No se recogen direcciones IP, ubicación ni datos sensibles. Tampoco se envía información a terceros.

Información adicional

No se utiliza la aplicación para realizar ningún tipo de seguimiento ni marketing. Toda la experiencia ocurre localmente en el dispositivo.

Niños

La aplicación no está dirigida a menores de 13 años. No se recopila intencionadamente información de niños. Si detectamos tal situación, eliminaremos los datos si fuera necesario.
